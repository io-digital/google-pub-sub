# Laravel Google Pub Sub

Library for Laravel to use the Google PubSub Methods

## Install

Via Composer

``` bash
$ composer require io-digital/google-pub-sub
```

## Usage

``` php
$googlePubSubHandler = new PubSubHandler();

OR

Dependency Inject: PubSubHandler
```
